Proctorio

Proctorio is an online, remote exam proctoring tool. This platform works with TITANium and Canvas for faculty and students.

Requirements for Faculty

  1. Install Google Chrome.
  2. Install the Proctorio extension   for Google Chrome.
  3. Navigate to your learning management system (Canvas or TITANium/Moodle) and create your exam.

Requirements for Students

Please set up Proctorio PRIOR to your exam date by following the instructions below. 

  1. Install Google Chrome  on a computer with a working webcam and microphone.
  2. Install the Proctorio extension for Google Chrome.
  3. Using Google Chrome, simply navigate to your learning management system (Canvas or TITANium/Moodle) and stat the proctored exam issued by your instructor.
