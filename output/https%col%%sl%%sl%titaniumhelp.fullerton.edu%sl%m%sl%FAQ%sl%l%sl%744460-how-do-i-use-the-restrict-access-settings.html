How do I use the Restrict access settings?

You will learn how to use the Restrict access settings which can be found when editing a Topic or Weekly section, an Activity or a Resource.

The Restrict access settings allow the instructor of a course in TITANium to decide exactly which user(s) can access or view any given Topic section, Activity or Resource in the course.

First, decide how you want to add a user restriction:

In a Topic/Weekly section

Using the Restrict access settings you can limit a student's access to a Topic or Weekly section in many different ways.

Here are some examples of ways you can use these settings:

  •  Hide a Topic section from all students except one
  • Set a Topic section to become visible on date
  • Hide a Topic section until a student has completed a specific activity
In an Activity or Resource

Using the Restrict access settings you can limit a student's access to an activity or resource in many different ways.

Here are some examples of ways you can use these settings:

  • Limit access to an activity using groups and restrict access
  • Set an Assignment for one specific group
  • Allow only a specific Grouping to view an activity

Article Summary

You have now learned some ways in which you can use the Restrict access settings in your course to limit or enable access to topic sections, activities and resources.
